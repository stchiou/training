    Item Code Item Name
    COURSEPGM-GBL-TLC-TECHWRIT Effective Technical Writing (English Verssion)
    COURSEPGM-GBL-TC_ERQAR-004I Event Report/Qality Assurance Report (ER/QAR) Workshop
    COURSEPGM-GBL-TC-ERQAR-007I Method 1 for Investigations
    COURSEPGM-GBL-TC-PHP-006 PHP Awareness (English Version)

    1. COURSEPGM-GBL-TC-ERQAR-004I and COURSEPGM-GBL-TC-ERQAR-007I are workshops that need to sign up through PLS. Credit for both courses will be granted once COURSEPGM-GBL-TC-ERQAR-004I is completed

    2. The assignment profiles in the CQ Lab that contain this curriculum are:

      • Laboratory Investigation Report Training (PGM-RMT-QTS_LIR)
      • QTS MIR Training for Responsible person and QA Approver (PGM-RMT-QTS_MIR_RP_APP)

    Compliance Specialist (PGM-RMT-QACQ_COMPL_SPEC)

    This assignment profile provides required training for accessing ExcelSafe (Compliant version of Excel Spreadsheet that keeps records of control substances). The following items are required for accessing ExcelSafe :

    1. ExcelSafe Access Form (USRM-FRM-00603, aka FORM-65483. The following are required information:
      • User Name field is the name of the requester
      • UID field is Hospira ID (GUID for employee after 2018)
      • Access Activity: check Add box
      • Access Level: check Data Entry
    2. Proof of completion of training:

      Provide a copy of Certificate of Completion printed out from PLS

    The form and the certificate should then submitted to Theresa Khelifi in Trailer I after obtaining Supervisor's signature.
